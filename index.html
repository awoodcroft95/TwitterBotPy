Twitterbotpy

Code for a blank Twitter Bot written in Python using Twython

View on GitHub Download .zip Download .tar.gz

What is this?

This is a simple automated twitter bot written in Python using the Twython Library.

What does it do?

This twitter bot will automatically search for tweets from a given criteria, then retweet the tweet(s) from that search. It will also favourite/like the tweet(s) and follow the account it was tweeted from. It also will follow all of your followers.

It does this once every 20 minutes(can be configured).
